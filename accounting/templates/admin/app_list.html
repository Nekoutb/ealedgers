{% comment %} EA Ledgers override of Django's admin/app_list.html. Stock Django renders each app's models as a vertical (one row per model). With ~22 accounting models that list is very long and hard to scan. This override renders each app's models as a responsive GRID of cards, so the whole model set forms a compact block (roughly a square) the user can take in at a glance — ~5 cards per row, wrapping into rows. Card = model name (links to its changelist) + small Add / Change actions.
